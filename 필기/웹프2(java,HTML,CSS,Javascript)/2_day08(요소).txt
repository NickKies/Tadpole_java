HTML 요소의 타입(Element type)

HTML의 모든 요소는 해당 요소가 웹 브라우저에 어떻게 보이는가를 결정짓는 display속성을 가집니다. 대부분의 HTML요소는 이러한 display 속성값으로 두가지 값중 하나를 가지게 되는데, 블록과 인라인 속성입니다.

1. 블록(block)
display속성값이 블록(block)인 요소는 언제나 새로운 라인(line)에서 시작하며, 해당 라인의 모든 너비를 차지합니다.
(<p>, <h>, <ul>, <ol>, <div>, <from>..등)

2. 인라인(inline)
display 속성값이 인라인(inline)인 요소는 새로운 라인(line)에서 시작하지 않습니다. 요소의 너비도 해당 라인 전체가 아닌 해당 html 요소의 내용(content)만큼만 차지합니다.
(<span>, <a>, <img>..등)

<div> 태그
다른 html 요소들을 하나로 묶는데 자주 사용되는 대표적인 블록 요소입니다. <div>요소는 주로 여러 요소들의 스타일을 한 번에 적용하기 위해 사용됩니다.

<span> 태그
텍스트(text)의 특정 부분을 묶는데 자주 사용되는 인라인 요소입니다. <span>요소는 주로 텍스트의 특정 부분에 따로 스타일을 적용하기 위해 사용됩니다.


아이프레임(iframe)

inline frame의 약자로 iframe 요소를 이용하면 해당 웹 페이지 안에 제한 없이 또 다른 하나의 웹페이지를 삽입할 수 있습니다.

문법
<iframe src="삽입할 페이지 주소" style="아이프레임의 크기를 설정할 CSS코드"></iframe>



<from> 태그
웹 페이지내에서 사용자로부터 입력을 받을 때 사용하는 태그입니다. 사용자가 입력한 데이터를 서버로 보낼 때 from요소를 사용합니다.

<from action="member_ok.jsp" method="" name="">
	입력 양식..
</from>

- action : 데이터를 보낼 파일위치
- method : 데이터를 전송하는 방법(get, post)
	  get => url, post=> header(보안)를 통해서 보냄 	
- name : 폼의 이름



<input> 태그
사용자로부터 데이터를 입력 받기 위해 사용되는 태그입니다.

<input type="text"> : 사용자로부터 글을 입력받는 글상자를 만듭니다.
size : 글상자의 크기, 값의 길이가 아닙니다.
maxlength : 값의 최대 길이를 설정합니다.
placeholder : 사용자가 입력 전에 특정 데이터를 입력 하도록 안내해 주는 값을 설정
readonly : 사용자가 입력 필드를 볼 수 있으나, 수정할 수 없게 설정(value값은 서버로 전송)
value : 설정된 값을 서버로 전송할 수 있습니다. 또한 미리 값을 설정할 수 있습니다.
required : 사용자가 데이터를 필수로 입력 하도록 설정합니다.
name : 요소의 이름을 설정합니다.


<input type="password"> : 사용자로부터 비밀번호를 입력받을 수 있습니다. 비밀번호를 입력받기 때문에 화면에는 입력받은 문자나 숫자 대신 별표나 작은 원으로 표현됩니다.


<input type="radio"> : 사용자로부터 여러 개의 옵션중에서 단 하나의 옵션만을 입력 받을 수 있습니다. 이때 서버로 정확한 입력을 전송하기 위해서는 반드시 모든 input 요소의 name속성이 같아야 합니다.
checked : 여러개의 옵션 중에서 처음에 미리 선택되는 옵션을 지정할 수 있습니다.


LEGEND요소
fieldset 요소의 제목(legend)을 표시합니다 fieldset 요소를 이용하여 여러 개의 컨트롤들을 묶었으면 이 묶음이 어떤 성격 또는 용도인지 알려줄 필요가 있으며, 이떄 legend요소를 사용합니다.

<input type="checkbox"> : 사용자로부터 여러 개의 옵션 중에서 다수의 옵션을 입력 받을 수 있씁니다. 체크박스는 라디오 버튼과는 달리 여러개의 옵션을 한 번에 입력 받을 수 있습니다. 이때 서버로 정확한 입력을 전송하기 위해서는 반드시 모든 input 요소의 name속성이 같아야 합니다.

<label> : 체크박스 뿐 아니라 텍스트를 클릭해도 체크박스에 체크가 됩니다.

*************************************************************************************************************
<input type="file"> : 사용자로부터 파일을 전송받을 수 있습니다. (파일 업로드시 ENCTYPE="multipart/form-data"속성을 꼭 입력해야 합니다.)
*************************************************************************************************************

<input type="color"> : 원하는 색상을 정의할 수 있습니다.
예) #00ff00 : 16진수 RGB컬러중 초록색
      -- -- --
     R   G  B

<input type="email"> : 타입이 email값일 때 전송하게 되면 input에 입력한 값이 이메일이 형식에 맞는지 체크하게 됩니다. "@"가 빠지면 입력창 하단에 풍선 도움말로 제대로 입력하지 않았다고 알려줍니다. 강제성은 없습니다.

<input type="url"> : 입력란에 http:// 가 앞자리에 와야 합니다. 값을 제대로 넣지 않으면 URL을 정확히 입력하라는 풍선 도움말을 보여줍니다. 강제성은 없습니다.

<input type="tell"> : 유효성 체크를 하지 않으며 자동으로 '-'을 넣어 주지는 않습니다. 각 나라마다 표준이 다르기 때문입니다. 이렇게 유효성 체크는 하지 않지만 모바일 기기에서 입력할 때 자동으로 숫자패드가 나타납니다. 강제성은 없습니다.

<input type="date"> : 원하는 날짜를 입력 할 수 있습니다.

<input type="number" min="" max="" step=""> : 사용자가 숫자를 입력 할 수 있습니다. number 타입이 일반 text 타입과 다른점은 입력 필드 우측에 숫자의 크기를 조절할 수 있는 상하 버튼이 생긴다는 점입니다. 

<input type="number" min="" max="" step="">
	                 -------   --------  ------
		    최소값   최대값  증가치



<input type="range"> : input 요소는 사용자가 일정 범위 안의 값만을 입력할 수 있도록 해줍니다.


<input type="search"> : 사용자가 검색어를 입력할 수 있도록 해줍니다. 이러한 검색 필드는 보통의 텍스트 필드와 동일하게 동작합니다. search 타입이 일반 text 타입과 다른 점은 입력 필드에 검색어를 입력하면, 입력 필드 우측에 입력된 검색어를 바로 삭제할 수 있는 x표시가 생기는 점입니다.


<input type="button">,<input type="submit">,<input type="reset"> : 사용자가 누를 수 있는 버튼을 나타냅니다. type의 속성값을 submit으로 설정하면 사용자로부터 입력받은 데이터를 서버의 폼 핸들러로 제출하는 버튼을 만들 수 있습니다. (폼핸들러 : 입력받은 데이터를 처맇기 위한 서버측의 웹 페이지를 의미합니다. 이러한 폼 핸들러의 주소는 form요소의 action속성을 이용하여 명시할 수 있습니다.) 버튼의 type값은 button(아무런 이벤트가 없는 버튼), submit(데이터를 전송하는 버튼), reset(입력양식을 초기화하는 버튼)이 있습니다.


선택 입력(SELECT)
select 요소는 여러개의 옵션이 드롭다운 리스트(drop-down list)로 되어있으며, 그중에서 단 하나의 옵션만을 입력받을 수 있습니다. option 요소는 드롭다운 리스트에서 선택할 수 있는 각각의 옵션을 명시합니다.

<select>
	<option value="실제값1">표현값1</option>
	<option value="실제값2">표현값2</option>

</select>


문장 입력(textarea)
사용자로부터 여러 줄의 텍스트를 입력받을 수 있습니다.

<textarea cols="가로 글자수 크기" rows="세로 글자수 크기">실제 값</text>

datalist
html5에 새로 추가된 속성으로 formㅇ소에서 미리 지정된 옵션 목록을 제공 합니다. 사용자들이 특정 글자를 입력하면 그에 해당하는 미리 지정된 목록을 보여 줍니다.

<input type="text" id="datalist id값">

<datalist id="data list의 id">
	<option value="실제값1" label="표기할 값1"></option>
	<option value="실제값2" label="표기할 값2"></option>
</datalist>


진행정도(progress)
진행정도를 나타낼 수 있씁니다. html5에서 추가되었고 IE버전 10부터 지원합니다.

<progress value="현재값" max="최대값"></progress>
