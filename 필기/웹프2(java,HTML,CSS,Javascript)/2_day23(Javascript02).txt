명시적 타입 변환 (explicit type conversion)
자바스크립트에서는 묵시적 타입변환을 많이 사용하지만, 명시적으로 타입을 변혼할 방법도 제공합니다. 명시적 타입 변환을 위해 자바스크립트가 제공하는 전역 함수는 다음과 같습니다.

1. Number()
2. String()
3. Boolean()
4. Object()
5. parseInt()
6. parseFloat()


산술 연산자(arithmetic operator)

산술 연산자는 사칙연산을 다루는 가장 기본적이면서도 많이 사용하는 연산자입니다.

+	왼쪽 피연산자의 값에 오른쪽 피연산자의 값을 더함.
-	왼쪽 피연산자의 값에서 오른쪽 피연산자의 값을 뺌.
*	왼쪽 피연산자의 값에 오른쪽 피연산자의 값을 곱함.
/	왼쪽 피연산자의 값을 오른쪽 피연산자의 값을 나눔.
%	왼쪽 피연산자의 값을 오른쪽 피연산자의 값을 나눈후, 그 나머지를 반환함.


대입 연산자(assignment operator)
대입 연산자는 변수에 값을 대입할 때 사용하는 ㅇ항 연산자 입니다.
=, +=, -=, *=, /=, %=

증감 연산자 (increment and decrement oprator)
증감 연산자는 피연산자를 1씩 증가 혹은 감소시킬 때 사용하는 연산자 입니다. 이 연산자는 피연산자가 단 하나뿐인 단항 연산자 입니다. 증감 연산자는 해당 연산자가 피연산자의 어느 쪽에 위치하는가에 따라 연산의 순서 및 결과가 달라집니다.

++x, x++, --x, x--


비교연산자
비교 연산자는 피연산자 사이의 상대적인 크기를 판단하여, 참과 거짓을 반환합니다.

==	값이 같다.
===	값이 같고 같은 타입니다.
!=	값이 같지 않으면 참.
!==	값이 같지 않거나 타입이 다르면 참.
>,>=,<,<=

논리 연산자
논리 연산자는 주어진 논리식을 판단하여, 참과 거짓을 반환합니다. && 연산자와 || 연산은 두개의 피연산자를 가지는 이항 연산자입니다.

&&	논리식이 모두 참이면 참을 반환
||	논리식 중에서 하나라도 참이면 참을 반환함
!	논리식의 결과가 참이면 거짓을, 거짓이면 참을 반환함

비트연산자
비트연산자는 논리 연산자와 비슷하지만, 비트 단위로 논리 연산을 수행합니다. 또한, 비트 단위로 전체 비틀ㄹ 왼쪽이나 오른쪽으로 이동시킬 때도 사용합니다.

&	대응되는 비트가 모두1 이면 1을 반환함
|	대응되는 비트 중에서 하나라도 1이면 1을 반환함
^	대응되는 비트가 서로 다르면 1을 반환함.
~	비트를 1이면 0으로, 0이면 1로 반전시킴.
<<	지정한 수만큼 비트를 전부 왼쪽으로 이동시킴
>>	부호를 유지하면서 지정한 수만큼 비트를 전부 오른쪽으로 이동시킴
>>>	지정한 수만큼 비트를 전부 오른쪽으로 이동시키며, 새로운 비트는 전부 0이 됨.


삼항 연산자
유일하게 피연산자를 세 개나 가지는 조건 연산자입니다.
표현식? 변환값1 : 변환값2


대화상자

1.alert()
window 객체의 alert() 메소드는 사용자에게 간단한 메시지를 보여주고, 그에 대한 사용자의 혹인을 기다립니다.

2. confirm()
사용자에게 간단한 메시지를 보여주고, 사용자가 확인이나 취소를 누리면 그 결과를 불리언 값으로 반환합니다.

3. prompt()
사용자에게 간단한 메시지를 보여주고, 사용자가 입력한 문자열을 반환합니다.

문제 

prompt() 메소드를 통해 이름, 국어점수, 영어점수, 수학점수를 입력 받아서 아래와ㅏ 같이 출력하는 프로그램을 작성

출력예시

이름 : 김사과
국어점수: 90
영어점수: 80
수학점수: 70
총점: 240점
평균 : 80점

제어문

프로그램의 순차적인 흐름을 제어해야 할 때 사용하는 실행문을 제어문이라고 합니다. 이러한 제어문에는 조건문, 반복문 등이 포함됩니다.

조건문
조건문이란 프로그램 내에세 주어진 표현식의 결과에 따라 별도의 명령을 수행하도록 제어하는 실행문 입니다.

1. if문
조건식의 결과가 참이면 주어진 실행문을 실행하며, 거짓이면 아무것도 실행하지 않습니다.

if(조건식) {
	조건식의 결과가 참일 때 실행하고자 하는 실행문
}
