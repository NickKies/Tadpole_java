<em>,<i> 태그
이탤릭체를 표현하고 싶을 때 <i>, <em> 태그를 사용합니다. <i>태그는 단순히 화면의 텍스트를 이탤릭체로 표현해줍니다. 하지만 <em>태그는 텍스트를 이탤릭체로 변환해줄 뿐만 아니라 그 내용이 중요하다는 의미도 함께 포함해 줍니다. 검색엔진은 <strong>태그나 <em>태그를 사용하여 강조된 텍스트를 더 중요하게 인식합니다.

<mark>태그
텍스트에 하이라이팅 효과를 적용시켜 줍니다.

<del>태그 
텍스트 중앙에 가로줄을 만들어 텍스트를 지운 것과 같은 효과를 내줍니다.

<ins>태그
텍스트 밑에 선을 그어 텍스트의 강조 효과를 내줍니다.

<sup>,<sub>태그
위첨자는 <sup>태그(superscript)를 사용하고, 아래첨자는 <sub>태그(subscript)를 사용하여 각각 표현할 수 있습니다.

1. 순서가 없는 리스트
<ul>태그로 시작을 하며, 여기에 포함되는 각가의 리스트 요소는 <li>태그로 시작합니다.
예) 
	<ul>
		<li>요소</li>
		<li>요소</li>
		...
	</ul>

2. 순서가 있는 리스트 
<ol>태그로 시작을 하며, 여기에 포함되는 각각의 리트스 요소는 <li>태그로 시작합니다.

type 속성의 값
1 : 숫자
A : 알파벳 대문자
a : 알파벳 소문자
i : 로마숫자
I : 로마숫자

start 속성의 값
ol태그에 사용하며 시작값을 임의로 설정할 때 사용합니다.
<ol type="속성의 값" start="start 속성의 값">
	<li>요소1</li>
	<li>요소2</li>
	...
</ol>


3. 정의 리스트
정의 리스트는 용어와 그에 대한 정의를 모아놓은 리스트로 <dl>태그로 시작합니다. <dt>태그에는 용어으 이름이 들어가고, <dd>태그에는 해당 용어에 대한 정의가 들어갑니다.

	<dl>
		<dt>제목</dt>
		<dd>내용 항목</dd>
		<dd>내용 항목</dd>
		...
	</dl>


이미지(image)
1. 웹 페이지에서 주로 사용된는 이미지의 종류
     1) JPEG 이미지 : .jpg .jpeg 휴대폰, 카메라 사진
     2) GIF 이미지 : .gif 움직이는 이미지(사진), 게임
     3) PNG 이미지 : .png 투명(알파)를 사용, 게임, 앱 => 뒤에 있는 이미지를 보임

2, html 문서에 이미지를 삽입할 때는 <img>태그를 사용합니다.

	<img src="이미지주소" alt="대체 문자열">

- 이미지 주소를 사용하는 방법
     1) 절대경로 : 고유한 경로. http:// 나 file://의 경로를 사용합니다.
     2) 상대경로 : 이미지를 삽입 할 HTML문서를 기준으로 경로를 인식합니다.

HTML 링크(Link)
웹페이지는 다른 페이지나 사이트로 연결되는 수많은 하이퍼링크(hyperlink)가 존재합니다. 이러한 하이퍼링크를 링크(link)라고 간단히 부르며, HTML에서는 <a>태그를 이용해 표현합니다.

<a href="문서의 위치">링크에 사용될 단어 또는 이미지</a>
             --------------   --------------------------------------
 	       1			2
- 1 : <a>태그의 href속성은 연결할 페이지 또는 사이트의 URL등을 명시합니다.
- 2 : <a>태그의 텍스트나 단락, 이미지등 다양한 HTML요소를 사용할 수 있습니다.

target 속성
<a>태그의 target 속성은 링크로 연결된 문서를 어디에서 열지를 명시합니다.

_blank : 링크로 연결된 문서를 새 창이나 새 탭에서 열어줍니다.
