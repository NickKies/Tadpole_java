CSS 컬러

1. 색상 이름으로 표현
aqua	black
blue	fuchsia
gray	green
lime	maroon
navy	olive
purple	red
silver	teal
white	yellow
...

예)
h2 { color: blue;}

2. RGB 색상값으로 표현
모니터나 스크린은 빨간색, 녹색, 파란색을 혼합하여 색을 표현합니다. HTML에서도 이 세가지 색을 가지고 색을 표현하는 RGB 색상을 사용합니다. RGB색상의 기본색은 각각 0부터 255까지의 범위 입니다.

예) 
h1 { color : rgb(0,200,0); }
h1 { color : rgba(0,200,0,0.5); }  /* alpha : 0 ~ 1 까지 투명도*/

3. 16진수 색상값으로 표현
16진수 색상값은 RGB 색상값을 각각 16진수로 변환한 것입니다. 따라서 RGB 색상의 기본색(빨초파)은 각각 00부터 ff까지의 범위를 가집니다.
h1 { color: #ff0000 ;} /* 대소문자를 구분하지 않음*/


CSS 텍스트

color 속성
텍스트의 색상을 설정합니다. 웹페이지에서 텍스트의 기본 색상은 검정색입니다. <body>태그에 명시된 color 속성값은 웹 페이지 내의 모든 텍스트 요송 적용됩니다. 하지만 각 요소별로 따로 명시된 color 속성값이 있다면, 해당 속성값이 <body>태그에 명시된 속성값보다 우선 적용됩니다.

letter-spacing 속성
텍스트 내에서 글자 사이의 간격을 설정합니다.

word-spacing 속성
텍스트 내에서 단어 사이의 간격을 설정합니다. letter-spacing 속성과는 달리 문자간 의 간격이 아닌 단어 간의 간격을 기준으로 설정합니다.


text-indent 속성
단락의 첫 줄에 들여쓰기 할지 안 할지를 설정합니다. 웹 페이지에서 단락은 기본적으로 들여쓰기가 설정되어 있지 않습니다.


text-align 속성
텍스트의 수평 방향 정렬을 설정합니다. text-align 속성으로 설정된 정렬 방향은 text-direction 속성과는 상관없이 우선적으로 적용됩니다.

text-decoration 속성
텍스트에 여러 가지 효과를 설정하거나 제거하는데 사용합니다.(line-through, overline, underline)

line-height 속성
텍스트의 줄 간격을 설정합니다.

text-shadow 속성
텐스트에 간단한 그림자 효과를 설정합니다.

text-shdow: 5px     5px       3px       #f00;
               -----     -----      -----       -----
           가로거리 세로거리 번짐정도  색상

번짐정도 : 그림자가 번지는 정도. 양수값을 사용하면 그림자가 모든 방향으로 퍼져 나가고 그림자가 크게 표시됩니다. 반대로 음수 값은 그림자가 모든 방향으로 축소되어 보입니다. 기본값은 0입니다.


text- transform 속성
텍스트에 포함된 영문자에 대한 대소문자를 설정합니다. 이 속성은 텍스트에 포함된 모든 영문자를 대문자나 소문자로 변경시켜 줍니다. 또한, 단어의 첫 문자만을 대문자로 변경시킬 수도 있습니다.


text-overflow 속성
텍스트가 기준선을 벗어나 넘칠 경우 넘치는 텍스트를 어떻게 처리할지 설정합니다.
clip : 넘치는 텍스트를 자릅니다. 
ellipsis : 말줄임표로 잘린 텍스트를 표시합니다.

white-space 속성
스페이스와 탭, 줄바꿈, 자동줄바꿈을 어떻게 처리할지 정하는 속성이빈다.
initial : 기본값으로 설정합니다. 
inherit : 부모 요소의 속성값을 상속받습니다.
normal : 기본값으로 글자 줄이 자동으로 바뀝니다. 콘텐츠가 요소의 너비를 초과할 경우 다음 줄로 바뀝니다.
nowrap : 줄바꿈이 실행되지 않습니다. 콘텐츠가 다음 줄로 바뀌지 않습니다.
pre : 줄 바꿈과 기타 공백이 유지딥니다. 이 가능한 값은 !DOCTYPE 선언에서 표준 준수 모드를 지정할 경우에 지원됩니다 표준 준수모드를 지정하지 않으면 이 값을 검색할 수는 있지만, 렌더링에 영향을 주지 않으며 normal 값처럼 동작합니다.
pre-line : 줄 바꿈 시퀀스가 유지됩니다.
pre-wrap : 줄 바꿈 시퀀스가 축소됩니다.


font-family 속성
하나의 글꼴만을 설정할 수도 있고, 여러개의 글꼴을 같이 설정할 수도 있습니다.
속성값이 여러 개의 글ㄲㄹ로 설정되어 있으면, 웹 브라우저는 위에서부터 순서대로 글꼴을 읽어 들입니다.

p { 
	font-family: 'Nanum Gothic', sans-serif;
	fint-size: 23px;
}
