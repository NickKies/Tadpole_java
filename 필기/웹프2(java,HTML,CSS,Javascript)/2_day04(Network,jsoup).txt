소켓(Socket)
각 포트를 사용하여 통신을 수행하는 도구. 각 프로그램에 포트를 세팅해 놓기만 하면 알아서
통신이 되는것이 아니라 소켓으로 데이터를 주고 받을 수 있게 하는 매체.

1. 서버 소켓과 클라이언트 소켓으로 나뉩니다.
2. 서버 소켓의 역할은 클라이언트 소켓의 연결 요청을 대기하고, 연결 요청이 오면 클라이언트 소켓을
생성하여 통신을 가능하게 합니다.
3. 클라이언트 소켓을 대기하는 것 없이 바로 사용가능하며, 실제로 데이터 송수신이 일어나는 것은 
클라이언트 소켓.


JSOUP(https://jsoup.org)
Jsoup은 자바로 만들어진 HTML파서(Parser)입니다. Jsoup은 자바 언어로 HTML을 다루기 쉽고
강력하게 파싱할 수 있는 기능을 제공합니다.

1. URL, 파일, 문자열을 소스로 하여 HTML을 파싱할 수 있습니다.
2. DOM 구조를 추적하거나 익숙한 CSS 선택자를 사용하여 데이터를 찾아 추출 할 수 있습니다.
3. 문서내의 HTML 요소, 속성, 텍스트를 조작할 수 있습니다.
4. 사용자가 입력한 데이터로 부터 XSS(Cross-Site Script) 공격을 방지하기 위해서 안전한 화이트 
리스트 방식으로 지정된 태그만 남기고 나머지는 제거할 수 있습니다. 
5. 깔끔한 형태의 HTML을 출력할 수 있습니다.
