반복문
반복문이란 특정 작업이 반복적으로 실행되도록 할 때 사용합니다.

1. while문
문법

while문은 소괄호 안에 문장이 true인 경우 중괄호가 감싸고 있는 문장을 반복하게 됩니다.

while(조건식) {
	반복 하고자 하는 실행문;
}

do~while문은 while문과 같지만 중괄호의 내용을 최소한 1번 실행하고 조건식을 판단한다는 것이 다릅니다.

do{
	반복 하고자 하는 실행문;
}while(조건식);


break문
break문은 switch문 또는 반복문을 벗어나는데 사용됩니다.

continue
continue문은 반복문의 끝으로 이동해 다음 반복으로 돌아갑니다. 전체 반복 중 특정 조건을 만족하는 경우만을 제외할 때 사용합니다.


2. for문

for(변수선언및초기화; 조건식; 증감식){
	반복하고자 할 문장;
}

for(int i=1; i<=10; i++){
	System.out.println("Hello World!");
}


다중 for문
다중 for문은 2개 이상의 for문이 겹쳐 사용되는 형태를 말합니다.

for(변수선언및초기화1; 조건식1; 증감식1){	// 5바퀴 돌기로 한다.!
	반복하고자 할 문장1;	// 총 5바퀴 돈다 !!
	for(변수선언및초기화2; 조건식2; 증감식2){	// 5바퀴 돌기로 한다!
		반복하고자 할 문장2; // 총 25바퀴 돈다 !!
	}
}
