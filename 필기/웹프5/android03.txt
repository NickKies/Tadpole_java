안드로이드 레이아웃 

1. 리니어 레이아웃
한 쪽 방향으로 차례대로 뷰를 추가하며 화면을 구성하는 방법

2. 프레임 레이아웃
- 가장 사우이에 있는 하나의 뷰 또는 뷰그룹만 보여주는 방법
- 여러 개의 뷰가 들어가면 중첩하여 쌓이게 됨
- 가장 단순하지만 여러개의 뷰를 중첩한 후 각 뷰를 전환하여 보여주는 방식으로 자주 사용됨

(AWT, Swing, JAVA FX)

3. 테이블 레이아웃
- 격자 모영의 배열을 사용하여 화면을 구성하는 방법
- HTML에서 많이 사용하는 정렬 방식과 유사하지만 안드로이드에서는 많이 사용하지 않음

4. 상대 레이아웃
부모 컨테이너나 다른 뷰와의 상대적 위치로 화면을 구성하는 방법

5. 제약 레이아웃
제약 조건을 사용해 화면을 구성하는 방법
안드로이드 스튜디오에서 자동으로 설정하는 디폴트 레이아웃
